### NodeJS-File-Hosting
Aiming for ShareX image/file servers

### Install
```bash
git clone gh repo clone krepowo/NodeJS-File-Hosting
cd NodeJS-File-Hosting
# edit config.json
npm install
node index.js
```

### Usage
Create a POST request with files to `/post`, example: `http://example.com?password=yourPassword`
